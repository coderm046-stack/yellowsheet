# Yellow Sheet PWA

**Student Exam Consolidator — Result Generator — PDF Slips**

## 🚀 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `yellow-sheet`)
2. Upload ALL these files to the **root** of the `main` branch:
   - `index.html`
   - `sw.js`
   - `manifest.json`
   - `maskable_icon_x192.png`
   - `maskable_icon_x512.png`
   - `maskable_icon.png`
3. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Wait ~60 seconds → your app is live at:
   `https://YOUR_USERNAME.github.io/REPO_NAME/`

## 📲 Install on Android

1. Open the live URL in **Chrome for Android**
2. Tap the **"Install App"** button in the header  
   — OR — tap Chrome menu → **"Add to Home Screen"**

## 🍎 Install on iPhone / iPad

1. Open the live URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**

## ✅ Files

| File | Purpose |
|------|---------|
| `index.html` | Main application |
| `sw.js` | Service Worker (offline support) |
| `manifest.json` | PWA manifest (icons, name, theme) |
| `maskable_icon_x192.png` | App icon 192×192 |
| `maskable_icon_x512.png` | App icon 512×512 |
| `maskable_icon.png` | App icon full size |
